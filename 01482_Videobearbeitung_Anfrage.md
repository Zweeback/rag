#### **1. Google Cloud Text-to-Speech API (kostenloses Kontingent)**
1. **Google Cloud Konto erstellen**:

   - Stelle sicher, dass du die **Text-to-Speech API** in der Cloud Console aktivierst.

2. **API-Key erstellen**:
   - Gehe zu **"APIs & Services"** und dann zu **"Credentials"**.
   - Erstelle einen neuen **API-Schlüssel**, den du später für die Verbindung zu Google TTS verwenden wirst.

3. **Installiere die Google Cloud-API-Bibliothek**:
   - Wenn du **Python** verwendest, installiere die **Google Cloud Client Library**:
     ```bash
     pip install google-cloud-texttospeech
     ```

4. **Erstelle dein Python-Skript**:
   - Erstelle ein Python-Skript, das Text in Sprache umwandelt. Beispiel:
     ```python
     from google.cloud import texttospeech

     # Initialisiere den TTS Client
     client = texttospeech.TextToSpeechClient()

     # Text, der in Sprache umgewandelt werden soll
     text = "Hallo! Wie geht es dir?"

     # Konfiguriere die Anfrage
     synthesis_input = texttospeech.SynthesisInput(text=text)
     voice = texttospeech.VoiceSelectionParams(
         language_code="de-DE", ssml_gender=texttospeech.SsmlVoiceGender.FEMALE
     )
     audio_config = texttospeech.AudioConfig(audio_encoding=texttospeech.AudioEncoding.MP3)

     # Anfrage ausführen
     response = client.synthesize_speech(
         input=synthesis_input, voice=voice, audio_config=audio_config
     )

     # Speichern der Audiodatei
     with open("output.mp3", "wb") as out:
         out.write(response.audio_content)

     print("Die Audiodatei wurde erfolgreich gespeichert!")
     ```

5. **API-Schlüssel verwenden**:
   - Stelle sicher, dass der **API-Schlüssel** in deiner Umgebung verfügbar ist (als Umgebungsvariable oder in einer Konfigurationsdatei), damit dein Skript auf die Google API zugreifen kann.
